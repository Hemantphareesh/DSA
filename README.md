1. Mathematics basics: Logarithms, exponents, modular arithmetic.
    **Laws of Exponents**
    - \(a^m \times a^n = a^{m+n}\)
    - \(\frac{a^m}{a^n} = a^{m-n}\)
    - \((a^m)^n = a^{m \cdot n}\)
    - \(a^0 = 1\) (for \(a \neq 0\))
